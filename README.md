# Laravel Project

## Requirements

- PHP >= 8.1
- Composer
- Node.js and npm 
- SQLite 
- Docker

## Installation

### Follow the steps below to set up the project on your local machine:

## Steps to run locally:
    git clone [https://github.com/jlfaelman/assessment-client](https://github.com/jlfaelman/assessment-api)
    cd assessment-api
    composer install
    npm install
    php artisan migrate
    php artisan serve
